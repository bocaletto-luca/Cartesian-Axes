About the Cartesian Coordinate System

The Cartesian coordinate system consists of two perpendicular axes: the horizontal x‑axis and vertical y‑axis, which intersect at the origin (0, 0). Points in this plane are represented as (x, y).

Quadrants:

  • Quadrant I: x > 0, y > 0
  • Quadrant II: x < 0, y > 0
  • Quadrant III: x < 0, y < 0
  • Quadrant IV: x > 0, y < 0

This tool allows you to plot points on a Cartesian plane and explore the coordinate system interactively.
